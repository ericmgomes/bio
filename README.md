# bio.ericgomes.me

Página estática mobile-first para GitHub Pages, baseada no perfil `gravatar.com/ericgomes120218`.

## Como editar

- Conteúdo e links: `index.html`
- Cores, layout compacto e responsividade: `styles.css`
- Interações pequenas: `script.js`
- Domínio customizado do GitHub Pages: `CNAME`

O site não depende de build, npm ou framework. Basta publicar estes arquivos no GitHub Pages.
